## Mockito

- Mockito is a mocking framework for unit tests in Java
- This repo contains educational materials covering the mocking concepts using Mockito.

### Slides
- A PDF document is included with an introduction on Mockito.

### Exercises
- 8  example unit test classes have been included for you to practices under the `src/test/java` directory
- 6 of which are examples, and 2 for you to complete on your own
- Please ensure you have all the dependencies included in your class path, refresh `build.gradle` file to get an update if needed
- Please note some tests have been designed to fail deliberately
- It is your responsibility to make them pass first - observe their behaviours and learn
